Dinosaur Feathers

Field Museum of Natural History, 2019
Role: UI Designer and Software Developer

A touch-screen interactive that guides visitors through an explanation of the cells responsible for feather colors in both birds and dinosaurs. This React application uses layered images, buttons, and sliders to create different interactive activities.
